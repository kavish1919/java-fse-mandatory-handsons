# Digital Nurture 5.0 – Java FSE Mandatory Hands-ons

A comprehensive collection of my solutions for the **Cognizant Digital Nurture 5.0 – Java Full Stack Engineer (Java FSE)** mandatory hands-on exercises.

This repository is intended to serve as a structured learning resource that demonstrates the practical application of Java, Design Patterns, Data Structures & Algorithms, PL/SQL, Unit Testing, Spring Framework, Microservices, React, and other technologies covered during the Digital Nurture Deep Skilling Program.

---

## Repository Goals

* Complete every mandatory hands-on exercise.
* Follow clean architecture and coding best practices.
* Write production-quality, readable, and maintainable code.
* Understand the underlying concepts instead of simply solving the exercises.
* Maintain a well-organized repository for future reference and interview preparation.

## Completed Modules

### Week 5: React
- **React-1**: Setup and first component using `create-react-app`.
- **React-2 (Student App)**: Functional components, JSX, and basic composition.
- **React-3 (Score Calculator)**: Props, arrow functions, and CSS stylesheets integration.
- **React-4 (Blog App)**: Class components, component state, lifecycle methods (`componentDidMount`, `componentDidCatch`), and Fetch API integration.
